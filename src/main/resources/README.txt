This directory should contain the following icon files for the Travel application:

1. travel-icon.png - Used by the JavaFX application as the window icon
2. travel-icon.ico - Used by Launch4j when creating a Windows executable

Please place appropriate travel-related icon files with these names in this directory.

For the PNG file, a size of 256x256 pixels is recommended.
For the ICO file, it should include multiple sizes (16x16, 32x32, 48x48, and 256x256) for best results with Windows.
